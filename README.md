# Weather App

**Real-Time Weather Forecast Application**

Weather App is a responsive web application that provides real-time weather information and a 5-day forecast using the OpenWeather API. The application allows users to search weather conditions by city name or automatically retrieve weather information based on their current location through the browser's Geolocation API.

---

## Features

* Real-time weather information
* Search weather by city name
* Current location weather using Geolocation API
* 5-day weather forecast
* Displays temperature, humidity, wind speed, and weather conditions
* Responsive user interface for desktop and mobile devices
* Error handling for invalid city names and API responses

## Project Overview

The application retrieves weather data from the OpenWeather API and presents it through a clean and intuitive user interface. It demonstrates the integration of third-party REST APIs, asynchronous JavaScript programming, DOM manipulation, and browser-based geolocation services.
